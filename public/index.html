Overview

Dev ภายนอกไม่ต้องเรียก Alfresco server โดยตรง ให้เรียกผ่าน API Gateway ตัวนี้เท่านั้น

External Dev / Postman
        |
        | Bearer token
        v
Node.js API Gateway
        |
        | Alfresco user/pass จาก .env
        v
Alfresco CMIS API

Base URL ตัวอย่าง: http://localhost:3000 ถ้า deploy หลัง Apache อาจเปลี่ยนเป็น URL ที่ผู้ดูแลระบบกำหนด เช่น http://your-domain/node-api

Authentication

ทุก endpoint ใต้ /api/alfresco/* ต้องส่ง Bearer token

Authorization: Bearer <API_TOKEN>

ห้าม hardcode token ใน frontend สาธารณะ ควรเก็บ token ใน backend หรือ environment variable ของระบบ dev ภายนอก

Endpoint Summary

Method Endpoint ใช้ทำอะไร
GET /api/health ตรวจว่า API Gateway ต่อ Alfresco ได้หรือไม่
GET /api/alfresco/folders ดูรายชื่อ folder ใต้ path ที่ระบุ
GET /api/alfresco/folders/files ดูไฟล์ทั้งหมดใต้ folder รวม subfolder
GET /api/alfresco/documents ดูไฟล์หรือค้นหาไฟล์ใน folder รวม subfolder
GET /api/alfresco/documents/:id/content เปิดหรือดาวน์โหลดไฟล์ด้วย document id

List Folders

GET/api/alfresco/folders?path=<folderPath>

ใช้ดูรายชื่อ folder ย่อยใต้ path ที่ส่งมา

Parameters

ParameterRequiredExampleความหมาย
pathNo/Sites/tg-saving/documentLibrarypath ของ folder ที่ต้องการดู folder ย่อย ถ้าไม่ส่งจะใช้ /

Example

GET http://localhost:3000/api/alfresco/folders?path=/Sites/tg-saving/documentLibrary
Authorization: Bearer <API_TOKEN>

List/Search Documents

GET/api/alfresco/documents?folderPath=<folderPath>&q=<keyword>

endpoint หลักที่แนะนำให้ dev ใช้ สำหรับ list และ search ไฟล์ใน folder รวม subfolder

Parameters

ParameterRequiredDefaultExampleความหมาย
folderPathNo/Sites/tg-saving/documentLibrary/การเงิน/Sites/tg-saving/documentLibrary/การเงินfolder path ที่ต้องการดึงหรือค้นหาไฟล์
pathNo-/Sites/tg-saving/documentLibrary/การเงินalias ของ folderPath
qNo-026277keyword สำหรับค้นจากชื่อไฟล์ ถ้าไม่ส่งจะ list ไฟล์ทั้งหมด
keywordNo-026277alias ของ q
nameNo-026277alias ของ q
maxItemsNo1000 list, 100 search100จำนวนรายการต่อหน้า
skipCountNo00จำนวนรายการที่ข้าม ใช้ทำ pagination

List Example

GET http://localhost:3000/api/alfresco/documents?folderPath=/Sites/tg-saving/documentLibrary/การเงิน&maxItems=100&skipCount=0
Authorization: Bearer <API_TOKEN>

Search Example

GET http://localhost:3000/api/alfresco/documents?folderPath=/Sites/tg-saving/documentLibrary/การเงิน&q=026277&maxItems=20&skipCount=0
Authorization: Bearer <API_TOKEN>

Open File Content

GET/api/alfresco/documents/:id/content?name=<fileName>

ใช้เปิดหรือดาวน์โหลดไฟล์จาก files[].id

ParameterRequiredExampleความหมาย
:idYes7b815e16-a594-4864-9665-cfda64e8d880%3B1.0document id จาก response ต้อง URL encode ถ้ามี ;
nameNofile.pdfชื่อไฟล์สำหรับ response header
 
GET http://localhost:3000/api/alfresco/documents/7b815e16-a594-4864-9665-cfda64e8d880%3B1.0/content?name=file.pdf
Authorization: Bearer <API_TOKEN>

Errors

{
  "message": "Cannot list Alfresco documents",
  "status": 500,
  "error": "connect ETIMEDOUT 172.17.1.21:80"
}
Statusสาเหตุวิธีตรวจสอบ
401ไม่ส่ง Bearer token หรือ token ผิดตรวจ header Authorization
403เรียก /dev-api/* จากเครื่องอื่นexternal dev ต้องใช้ /api/alfresco/*
400parameter ไม่ครบหรือ document id ไม่ถูกต้องตรวจ URL/path/query
404path หรือ id ไม่พบใน Alfrescoตรวจ folderPath หรือ document id
500API Gateway ต่อ Alfresco ไม่ได้ หรือ Alfresco errorเช็ก /api/health และ Alfresco server
